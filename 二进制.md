## 只出现一次的数字
```java
var singleNumber = function(nums) {
  let res = 0;
  nums.forEach(e => {
    res = res ^ e;
  })
  // res 就是 a ^ b 
  let right = res & (~res + 1);
  // a和b必然最后某一位不同，一个是0，一个是1
  let res1 = 0;
  nums.forEach(e => {
    if((e & right) === 0){
      res1 ^= e;
    }
  })
  return [res1, res ^ res1]
};

```

## 不使用加减使用两数之和
- a 加 b，包括两部分，位不同的 + 进位的
- 位不同的就是a ^ b
- 位相同的，进位的就是 a & b >> 1
- 递归，因为a & b 总会变成0
```java
public int getSum(int a, int b) {
  int sum = a ^ b;
  int temp = (a & b) << 1;
  if (temp != 0) {
      return getSum(sum, temp);
  }
  return sum;
}
```
