# 数学
## 构建乘积数组 

```java
class Solution {
    public int[] constructArr(int[] a) {
        int len = a.length;
        if(len == 0) return new int[0];
        int[] b = new int[len];
        b[0] = 1;
        int tmp = 1;
        for(int i = 1; i < len; i++) {
            b[i] = b[i - 1] * a[i - 1];
        }
        for(int i = len - 2; i >= 0; i--) {
            tmp *= a[i + 1];
            b[i] *= tmp;
        }
        return b;
    }
}
```

## 第n个丑数
使用3指针加动态规划

```java
class Solution {
    public int nthUglyNumber(int n) {
        int n2 = 0,n3 = 0,n5 = 0;
        int[] dp = new int[n];
        dp[0] = 1;
        for(int i = 1;i<n;i++){
            dp[i] = Math.min(2*dp[n2],Math.min(3*dp[n3],5*dp[n5]));
            if(dp[i] == 2*dp[n2]) n2++;
            if(dp[i] == 3*dp[n3]) n3++;
            if(dp[i] == 5*dp[n5]) n5++;
        }
        return dp[n-1];
    }
}
```
